# Chrome Web Store Listing Copy

## Extension Name
DockStack

## Short Description
Local-first DevTools-grade capture, extraction, and export workspace for modern web apps.

## Promotional Description
DockStack helps you inspect what sites actually load, review structured network data, and turn useful responses into local datasets and exportable artifacts.

## Full Description
DockStack is a local-first browser workspace built for serious capture and extraction workflows.

Instead of focusing only on page HTML, DockStack helps you work with the real sources modern applications rely on:
- fetch requests
- XHR responses
- JSON payloads
- GraphQL-style data shapes
- rendered application views

DockStack includes:
- capture session controls
- request detail inspection
- structured dataset detection
- dataset previews
- JSON and CSV export paths
- local-native integration for heavier processing
- local AI hooks for analysis workflows

DockStack is designed for developer, analyst, and technical inspection workflows where visibility and local control matter.

## Core Features
- Start and stop local capture sessions
- Scope capture by tab, domain, or workspace
- Inspect request and response details
- Detect structured JSON dataset candidates
- Preview extracted rows before export
- Export session data to JSON or CSV
- Route heavier work into a local native core

## Permissions Justification Summary
DockStack requests advanced permissions because it is a capture and inspection tool, not a decorative browser enhancement.

It uses those permissions for:
- local capture controls
- page interaction
- export
- native messaging
- advanced local processing

A detailed permission breakdown should be linked from the listing or support site.

## Privacy Disclosure Summary
DockStack is intended to operate as a local-first tool. Sensitive capture mode is explicit and opt-in. Known secret-bearing headers are masked before persistence. Users should only use DockStack in environments where they are authorized to inspect the data being captured.
